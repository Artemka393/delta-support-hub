# SupportFlow Hub

SupportFlow Hub - это небольшой внутренний сервис для команды поддержки. Он помогает принимать обращения, автоматически определять приоритет и подсказывать специалисту следующий шаг.

## Выполняемые задачи

- Создание заявки пользователем и автоматическую классификацию.
- Очередь заявок с фильтрацией по статусам и SLA.
- Связь frontend и backend через REST API.
- EF Core модели под SQL Server.
- Seed-данные для быстрого локального запуска.
- Авторизацию через корпоративный SSO.
- Историю комментариев и действий по заявке.
- Интеграцию с почтой или мессенджером.
- Полнотекстовый поиск по базе знаний.
- Метрики качества поддержки: среднее время реакции, просрочки SLA, повторные обращения.
